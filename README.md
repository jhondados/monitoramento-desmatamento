# 🌱 Monitoramento de Desmatamento

> Monitoramento de desmatamento com satélite e IA. Detecta supressão de vegetação para compliance ambiental e crédito de carbono.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/monitoramento-desmatamento/

[![Vertical](https://img.shields.io/badge/Vertical-Agro-blue)]() [![IA](https://img.shields.io/badge/IA-Computer%20Vision-green)]() [![Demo](https://img.shields.io/badge/Demo-Interativa-orange)]()

## 📊 Resultados-chave (KPIs)
**97.8%** Precisão · **5M km²** Área · **16 dias** Alertas · **estimado** Carbono

## 🧠 Tecnologias de IA
Computer Vision · Satellite · ML

## ✨ O que faz
- Detecção com Sentinel-2/INPE
- Alertas DETER (16 dias)
- Estimativa de carbono
- Compliance ambiental

## 🗄️ Fonte de dados real
[INPE PRODES/DETER](https://terrabrasilis.dpi.inpe.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com INPE PRODES/DETER
2. **Processamento com IA** — Computer Vision · Satellite · ML
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
